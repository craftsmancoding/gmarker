The Gmap Snippet lets you easily draw a Google map using flexible address information or latitude and longitude coordinates.

image::http://maps.googleapis.com/maps/api/staticmap?center=-12.043333,-77.028333&size=610x300&sensor=true&zoom=15[Standard map]

See this video for a quick overview: http://youtu.be/13gK6jiZ1RI

== Parameters == &div (string) optional CSS id of the target div. Default: map. &headTpl (string) optional Chunk name containing the necessary JS and CSS to inject into the page head. Default: gmapshead. &latlng (string) latitude,longitude coordinates (comma-separated). Overrides &address. &address (string) what you might type into a Google map search. Defaults to the gmarker.format_string, which in turn inherits values from the page on which the Snippet appears. &type ROADMAP | TERRAIN | SATELLITE the type of map you wish to generate. Default: ROADMAP &height (string) desired map height (specify px or % -- v1 was pixels only). Defaults to System Setting gmarker.default_height. &width (string) desired map width (specify px or % -- v1 was pixels only). Defaults to System Setting gmarker.default_width. &static (boolean) set this to 1 if want your map to be fixed. Default is 0. TODO: NOT YET INTEGRATED &zoom* (integer) default is 15.

== Examples ==

=== Simplest Example ===

The "simplest" example requires that you have correctly configured the Extra and its System Settings.

[[Gmap]]

This invocation of the Snippet with no parameters is identical to this:

[[Gmap? &address=[[++gmarker.format_string]] &height=[[++gmarker.default_height]] &width=[[++gmarker.default_width]]]]

If your gmarker.format_string is set to something like the following:

[[address]], [[city]], [[*state]]

Then the Snippet call is equivalent to this:

[[Gmap? &address=[[*address]], [[*city]], [[*state]] &height=[[++gmarker.default_height]] &width=[[++gmarker.default_width]]]]

That would mean that the page where you put this Snippet call must have values set for [[address]], [[city]], and [[*state]]. If your page does not have TVs of that name, then the &address parameter will evaluate to nothing and the map won't work!

=== Manual Execution ===

To use the Gmaps Snippet with no prior configuration, you must supply enough parameters to generate a map:

[[Gmap? &lat=-12.043333 &lng=-77.028333 &height=400 &width=800]]

Or rely on the https://github.com/craftsmancoding/gmarker/wiki/System-Settings[System Settings] defaults to specify the height and width:

[[Gmap? &lat=-12.043333 &lng=-77.028333]]

Or you can pass an address of your own choosing directly to the Snippet:

[[Gmap? &address=221 B Baker St. London, England]]
